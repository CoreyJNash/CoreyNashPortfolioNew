Joke Book

Joke Book is a full stack web app that lets users share and read jokes made b signed in users. You can log in to create, update and delete jokes. Or you can just visit the site to view jokes written by others. This app was created using Ruby on Rails and Heroku for the backend. The front end was made using HTML/CSS, Javascript and JQUERY.

Top Shelf Board Games

This is a full stack web app that allows a user to log in and see a catalog of board games shared between a community of users. They can sign in to create new games to put on the list, update ones that are already there and delete games they no longer have or don’t want to be seen publicly. This site was made using a few different resources including for the backend Ruby/Ruby on Rails/PSQL/HEROKU and for the FrontEnd using HTML/CSS and Javascript

Tic-Tac-Toe Extreme

Welcome to Tic-Tac-Toe Extreme edition. This is a fully functional tic-tac-toe web app with full game logic and pre-built API on Heroku. A user can either just play a game with a friend localy or they can sign in, create a game and see a log of past games played by the user.
